## Рекомендации по интеграции ресурсов в пайплайн

### Автоматизация верификации

1. **Tanzil.net API** — автоматическая проверка коранических аятов (номер суры + аята → текст)
2. **Sunnah.com API** — автоматическая верификация хадисов по номеру сборника
3. **Shamela.ws** — полнотекстовый поиск для сверки цитат

### Порядок использования при верификации сегмента

```
1. Коранические аяты     → Quran.com / Tanzil.net (автомат.)
2. Хадисы (основные)     → Sunnah.com (автомат.) → Dorar.net (степень)
3. Хадисы (редкие)       → Dorar.net → Shamela (полнотекстовый поиск)
4. Иснады                → Тахзиб ат-Тахзиб (Shamela)
5. Имена и биографии     → Аль-Исаба / Сияр ан-Нубаля (Shamela)
6. География             → Му'джам аль-Бульдан (Shamela)
7. Даты                  → IslamicFinder Calendar
8. Терминология (RU)     → IslamQA.info (рус.) + Islam-today.ru
9. Терминология (UZ)     → Islom.uz + IslamQA.info (узб.)
10. Терминология (TJ)    → Публикации Муфтията ТД
```
