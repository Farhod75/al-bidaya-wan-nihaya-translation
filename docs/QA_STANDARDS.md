## Принципы (адаптированы из hadith-verifier)

| Принцип в hadith-verifier | Адаптация для проекта перевода |
|---------------------------|-------------------------------|
| AI flags, human decides | Агенты переводят и проверяют, человек утверждает |
| One concern per commit | Один том / одна глава на коммит |
| Documentation is not optional | Каждый том сопровождается TRANSLATION_LOG |
| Read before write | Агент сверяется с глоссарием и предыдущими томами перед работой |
| CI GREEN GATE | Финальная проверка QA должна быть ✅ перед публикацией |

---

## Три уровня требований

**MUST** = обязательно, блокирует публикацию  
**SHOULD** = по умолчанию, отступать только с обоснованием  
**OPTIONAL** = в зависимости от контекста

## ЭТАП 1: Предварительная подготовка (до начала перевода)
## ════════════════════════════════════════════════════════

### Чек-лист подготовки

- [ ] **MUST** — Арабский оригинал главы загружен и верифицирован (Дар Хаджар / ат-Турки)
- [ ] **MUST** — Английский перевод Darussalam загружен для кросс-сверки
- [ ] **MUST** — Глоссарий терминов обновлён до текущей главы
- [ ] **SHOULD** — Контекст предыдущей главы передан агенту для связности
- [ ] **SHOULD** — Список имён собственных и географических названий в главе выделен
- [ ] **OPTIONAL** — Существующие русские переводы глав собраны для сравнения

## ════════════════════════════════════════════════════════
## ЭТАП 2: Первичный перевод
## ════════════════════════════════════════════════════════

### Чек-лист первичного перевода

- [ ] **MUST** — Все аяты Корана приведены в общепринятом переводе (Кулиев / Саблуков / Крачковский для РУ)
- [ ] **MUST** — Хадисы даны с указанием источника (сборник + номер)
- [ ] **MUST** — Имена собственные транслитерированы согласно глоссарию
- [ ] **MUST** — Географические названия унифицированы
- [ ] **MUST** — Даты хиджры сопровождаются григорианским эквивалентом
- [ ] **SHOULD** — Стиль изложения соответствует контексту (нарратив / научный)
- [ ] **SHOULD** — Сноски и комментарии автора сохранены и помечены
- [ ] **OPTIONAL** — Дополнительные пояснения переводчика отмечены [Прим. перев.]

### Метрики оценки первичного перевода

| Метрика | Описание | Порог прохождения |
|---------|----------|-------------------|
| **Полнота** | % переведённых абзацев от общего числа | ≥ 100% |
| **Терминологическая консистентность** | % терминов, совпадающих с глоссарием | ≥ 95% |
| **Точность аятов** | Все аяты сверены с эталонным переводом | 100% |
| **Точность хадисов** | Указан источник и номер для каждого хадиса | 100% |
| **Читаемость** | Оценка по шкале 1-5 (редактор) | ≥ 3.5 |

## ════════════════════════════════════════════════════════
## ЭТАП 3: Редакторская правка
## ════════════════════════════════════════════════════════

### Чек-лист редактуры

- [ ] **MUST** — Стилистическое единообразие с предыдущими главами
- [ ] **MUST** — Отсутствие неоднозначных толкований вероучительных моментов
- [ ] **MUST** — Все [Прим. перев.] проверены на корректность
- [ ] **MUST** — Пунктуация и орфография проверены
- [ ] **SHOULD** — Длина предложений ≤ 40 слов (исключение: цитаты)
- [ ] **SHOULD** — Абзацы ≤ 200 слов
- [ ] **SHOULD** — Связность между абзацами (переходные фразы)
- [ ] **OPTIONAL** — Альтернативные переводы спорных мест вынесены в сноски

### Автоматизированные проверки (MUST)

```
Проверка                          | Инструмент           | Действие при ошибке
─────────────────────────────────────────────────────────────────────────────
Пропущенные абзацы               | Скрипт сравнения     | БЛОКИРУЕТ публикацию
Дублированные абзацы             | diff/hash            | БЛОКИРУЕТ публикацию
Непереведённый арабский текст    | Regex [\u0600-\u06FF]| Пометить для проверки
Отсутствие ссылки на хадис       | Regex-паттерн        | БЛОКИРУЕТ публикацию
Несоответствие глоссарию         | Скрипт проверки      | Предупреждение
Нарушение формата дат            | Regex х./г.          | Предупреждение
Длина предложений > 40 слов     | word count           | Предупреждение
Битые Unicode символы            | encoding check       | БЛОКИРУЕТ публикацию
```

## ЭТАП 4: Верификация исламского контента
## ════════════════════════════════════════════════════════

### Чек-лист верификации

- [ ] **MUST** — Аяты Корана: номер суры и аята верен
- [ ] **MUST** — Хадисы: источник существует и номер корректен
- [ ] **MUST** — Классификация хадисов (сахих/хасан/даиф) указана, если присутствует в оригинале
- [ ] **MUST** — Имена сподвижников с (разия Ллаху анху/анха) после каждого упоминания
- [ ] **MUST** — Имя Пророка ﷺ с салаватом при каждом упоминании
- [ ] **MUST** — Нет искажений вероучительных позиций ахлю-с-сунна валь-джамаа
- [ ] **SHOULD** — Иснады хадисов сверены с базой sunnah.com / dorar.net
- [ ] **SHOULD** — Биографические данные (имена, даты) сверены с «Сияр а'лям ан-нубаля» (аз-Захаби)
- [ ] **OPTIONAL** — Примечания ат-Турки (мухаккик) отражены

### Уровни серьёзности ошибок (адаптация Severity из hadith-verifier)

| Уровень | Описание | Примеры | Действие |
|---------|----------|---------|----------|
| **CRITICAL** | Искажение вероубеждения, неверная атрибуция аята/хадиса | Хадис приписан не тому сборнику; аят Корана переведён с ошибкой в смысле | ❌ БЛОКИРУЕТ публикацию. Возврат на этап перевода |
| **HIGH** | Пропуск значимого контента, неверная дата | Пропущен абзац; дата хиджры/григорианская не совпадает | ❌ БЛОКИРУЕТ публикацию. Исправление обязательно |
| **MEDIUM** | Стилистические несоответствия, мелкие терминологические расхождения | Разная транслитерация одного имени; стиль сноски отличается | ⚠️ Исправить до публикации, не блокирует |
| **LOW** | Косметические проблемы | Лишний пробел; пунктуация | 📝 Исправить при следующей ревизии |

## ЭТАП 5: Языковая проверка (для каждого целевого языка)
## ════════════════════════════════════════════════════════

### Русский язык

- [ ] **MUST** — Кириллический текст без латинских символов (кроме цитат)
- [ ] **MUST** — Орфография проверена (aspell/hunspell ru_RU)
- [ ] **MUST** — Богословские термины в согласии с традицией русскоязычного исламоведения
- [ ] **SHOULD** — Стиль: научно-популярный, доступный образованному читателю
- [ ] **SHOULD** — Нет калек с арабского/английского языка

### Узбекский язык (кириллица)

- [ ] **MUST** — Весь текст в узбекской кириллице (ғ, ҳ, қ, ў, ҳ)
- [ ] **MUST** — Нет смешения латинского и кириллического узбекского
- [ ] **MUST** — Нет таджикских слов в узбекском тексте (P003 из hadith-verifier)
- [ ] **SHOULD** — Терминология согласована с существующими узбекскими исламскими изданиями

### Таджикский язык

- [ ] **MUST** — Весь текст в таджикской кириллице
- [ ] **MUST** — Нет узбекских слов (P003 из hadith-verifier: «ташриф буюринг» → «барои дидан гузаред»)
- [ ] **MUST** — Специфические таджикские буквы (ғ, ӯ, ҳ, ҷ, ӣ) используются корректно
- [ ] **SHOULD** — Стиль согласован с существующими таджикскими переводами

### Автоматизированная языковая проверка

```python
# Паттерны для автоматической проверки (адаптация из hadith-verifier)

LANGUAGE_CHECKS = {
    'ru': {
        'must_contain': r'[\u0400-\u04FF]',           # Кириллица
        'must_not_contain': r'[\u0600-\u06FF]{10,}',  # Блоки арабского (>10 символов подряд)
        'script_name': 'Cyrillic',
    },
    'uz_cyrillic': {
        'must_contain': r'[ғҳқўҲҚ]',                 # Узбекские спецсимволы
        'must_not_contain': r'[ӯҷӣ]',                 # Таджикские символы
        'script_name': 'Uzbek Cyrillic',
    },
    'tj': {
        'must_contain': r'[ғӯҳҷӣ]',                   # Таджикские спецсимволы
        'must_not_contain': r'[қў]',                   # Узбекские символы
        'banned_phrases': ['ташриф буюринг', 'марҳамат қилинг'],  # P003
        'script_name': 'Tajik Cyrillic',
    },
}
```

## ЭТАП 6: Финальная проверка перед публикацией
## ════════════════════════════════════════════════════════

### Финальный чек-лист (QA Gate)

- [ ] **MUST** — Все ошибки уровня CRITICAL и HIGH исправлены
- [ ] **MUST** — Верификатор исламского контента дал одобрение
- [ ] **MUST** — Автоматизированные проверки прошли без ошибок
- [ ] **MUST** — Файл сохранён в правильном формате (UTF-8 без BOM — P020)
- [ ] **MUST** — Нейминг файла соответствует конвенции (GITHUB_WORKFLOW.md)
- [ ] **SHOULD** — Перекрёстная проверка: другой языковой агент проверил перевод
- [ ] **SHOULD** — TRANSLATION_LOG.md обновлён для данной главы
- [ ] **OPTIONAL** — Аудио-разметка готова для озвучки

### Критерии прохождения / провала (Pass/Fail)

| Критерий | Pass | Fail |
|----------|------|------|
| Полнота перевода | 100% абзацев переведены | Любой пропущенный абзац |
| Аяты Корана | Все сверены, все корректны | Хоть один неверный аят |
| Хадисы | Все с источниками, все верифицированы | Хоть один без источника |
| Терминология | ≥ 95% совпадение с глоссарием | < 90% совпадение |
| CRITICAL ошибки | 0 | Любая ≥ 1 |
| HIGH ошибки | 0 | Любая ≥ 1 |
| MEDIUM ошибки | ≤ 3 на главу | > 5 на главу |
| Автоматизированные проверки | Все ✅ | Любая блокирующая ❌ |

## ════════════════════════════════════════════════════════

### Метрики по тому (отслеживаются в TRANSLATION_LOG)

| Метрика | Как измеряется | Целевое значение |
|---------|---------------|-----------------|
| **Скорость перевода** | Страниц/день | ≥ 5 стр/день |
| **Время на ревизию** | Часов на главу | ≤ 2 часа |
| **Ошибки верификации** | Кол-во CRITICAL+HIGH на главу | 0 после 2-й итерации |
| **Терминологическая точность** | % совпадения с глоссарием | ≥ 97% |
| **Читаемость (редактор)** | Шкала 1-5 | ≥ 4.0 |
| **Консистентность между языками** | Дельта объёма RU vs UZ vs TJ | ≤ 15% |
| **Время до публикации** | Дней от начала перевода до публикации | ≤ 14 дней/глава |

### Метрики качества AI-агентов (адаптация Scoring Rubric из hadith-verifier)

| Оценка | Значение для перевода |
|--------|----------------------|
| **1.0** | Перевод точен, стилистически безупречен, терминология верна, готов к публикации |
| **0.7** | Незначительные стилистические замечания, суть передана верно, требует минимальной правки |
| **0.4** | Заметные ошибки в терминологии или стиле, требует серьёзной правки |
| **0.0** | Искажение смысла, пропуски, недопустимые ошибки в вероучительных моментах |

## АВТОМАТИЗИРОВАННЫЕ ПРОВЕРКИ (Pipeline)

### Скрипты проверки (запускаются при каждом коммите)

```yaml
# Порядок выполнения (адаптация CI test run order из hadith-verifier)

1. encoding_check      — UTF-8, без BOM, без битых символов        [FAST]
2. completeness_check  — все абзацы оригинала имеют перевод         [FAST]
3. glossary_check      — терминологическое соответствие             [FAST]
4. quran_verify        — аяты сверены с эталоном                    [MEDIUM]
5. hadith_verify       — хадисы имеют источники                     [MEDIUM]
6. language_check      — правильный скрипт, нет языкового дрифта   [FAST]
7. style_check         — длина предложений, абзацев                [FAST]
8. cross_reference     — перекрёстная проверка между языками       [SLOW]
```

## LLM-КАК-СУДЬЯ (адаптация LLM-as-Judge из hadith-verifier)
## ════════════════════════════════════════════════════════

### Когда использовать
- После первичного перевода — оценка качества перед отправкой редактору
- После редактуры — сравнение «до/после»
- Периодическая проверка консистентности между главами

### Промпт для оценки перевода

```
Ты — эксперт-оценщик перевода исламских текстов с арабского.
Оцени перевод по шкале 0–1.0.

Критерии:
- Точность (accuracy): смысл оригинала передан верно
- Полнота (completeness): ничего не пропущено
- Терминология (terminology): богословские термины корректны
- Стиль (style): читабельно, соответствует жанру
- Консистентность (consistency): согласовано с глоссарием

Оригинал (арабский): {original}
Перевод: {translation}
Глоссарий: {glossary_excerpt}

Верни только JSON:
{
  "accuracy": 0.0,
  "completeness": 0.0,
  "terminology": 0.0,
  "style": 0.0,
  "consistency": 0.0,
  "overall": 0.0,
  "issues": ["строка"],
  "suggestions": ["строка"]
}
```
