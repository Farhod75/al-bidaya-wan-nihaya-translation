## 🌟 О проекте

**«Аль-Бидая ва-н-Нихая»** (البداية والنهاية) — фундаментальный историко-богословский труд имама Ибн Касира (1301–1373 гг.), охватывающий историю от сотворения мира до событий эсхатологии. Это один из важнейших источников исламской историографии.

### 🎯 Цели проекта

- **Качественный перевод** на три языка с сохранением смысловой точности
- **Единая терминология** через общий глоссарий исламских терминов
- **Аудиокнига** для слабовидящих и незрячих людей — садака джария (непрерывное благодеяние)
- **Открытый исходный код** для вклада сообщества

### 📚 Языки перевода

| Язык | Статус | Папка |
|------|--------|-------|
| 🇷🇺 Русский | 🟡 В процессе | `translation/ru/` |
| 🇺🇿 Узбекский (кириллица) | 🟡 Планируется | `translation/uz/` |
| 🇹🇯 Таджикский | 🟡 Планируется | `translation/tj/` |

## 🚀 Быстрый старт

### Для переводчиков

1. Ознакомьтесь с [Конституцией проекта](docs/PROJECT_CONSTITUTION.md)
2. Прочитайте [Правила участия](CONTRIBUTING.md)
3. Изучите [Структуру агентов](docs/AGENTS_STRUCTURE.md) для понимания процесса
4. Посмотрите [Стандарты качества](docs/QA_STANDARDS.md)

### Для разработчиков

```bash
# Клонировать репозиторий
git clone https://github.com/Farhod75/al-bidaya-wan-nihaya-translation.git
cd al-bidaya-wan-nihaya-translation

# Создать ветку для работы
git checkout -b translate/vol01-ru-chapters-1-5

# После работы — пушить и создать PR
git push origin translate/vol01-ru-chapters-1-5
```
